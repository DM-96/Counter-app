# Counter Dinamico

## Descrizione
Una semplice applicazione web che simula un contatore. Permette di aumentare o diminuire il valore tramite due pulsanti creati dinamicamente con JavaScript puro.

## Funzionalità
- Valore iniziale del contatore = 0
- Pulsante "+" per incrementare
- Pulsante "−" per decrementare
- Aggiornamento dinamico del display
